## js的数据类型
 ####  1. 基本类型
  - number
  - string
  - boolean
  - undefined
  - null
  - symbol

#### 2. 复杂类型
  - object


## 数据类型的判断
  - `typeof 1 ==>number`
  - `typeof 'hello' ==> string`
  - `typeof true ==> boolean`
  - `typeof undefined ==> undefined`
  - `typeof null ==> object`
  - `typeof [] ==> object`
  - `typeof function(){} ==> function`
  - `typeof {} ==> object`
